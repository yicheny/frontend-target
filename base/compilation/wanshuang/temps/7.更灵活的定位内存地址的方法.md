# ASCII
ascii是一套通用编码方案或者说编码表

从键盘输入`a`，在屏幕显示`a`，过程是怎样的？
1. 按下`a`键，输入信息被传递给计算机
2. 计算机将其编码成ascii码`61h`保存在指定内存空间中
3. 文本编辑器从内存中取出编码`61h`，将其送到显卡的显存上
4. 工作在文本模式下的显卡，用`ascii`码解析显存内容，`61h`会被作为字符`a`处理
5. 显卡驱动显示器，将字符`a`的图像画到屏幕上
