# 后序
## 递归实现
```c++
template <typename T, typename VST>
void travPre_R ( BinNodePosi(T) x, VST& visit ) { 
   if ( !x ) return; 
   travPre_R ( x->lc, visit ); 
   travPre_R ( x->rc, visit ); 
   visit ( x->data ); 
} 
```

## 迭代实现
后序遍历是自底而上的，思路如下：
0. 将其分解为若若干个片段
1. 访问当前节点
2. 遍历其右兄弟（若存在）为根的子树
3. 向上回溯至父节点（若若存在）转入下一片段
