# 什么是CSS的值？
CSS的值是一种定义**允许子值**集合的方法【亦称CSS数据类型】。

通常以`<>`形式包裹，比如`<length>`,`<color>`，虽然和`html`元素形式有些类似，但是因为两者使用的上下文不同，所以很难被混淆。

# 数值数据类型
CSS中使用了各种数值数据类型，见：

| 数据类型 | 描述 | 
| -- | -- |
| `<integer>` | 整数，比如1024或5 | 
| `<number>` | 小数，可以有小数部分也可以没有，比如128、0.255、-1.3
| `<dimension>` | `<dimension>`是一个`<number>`，它有一个附加的单位，例如`45deg`、`5s`或`10px`。`<dimension>`是一个伞形类别，包括`<length>`、`<angle>`、`<time>`和`<resolution>`类型。

## 长度`<length>`
`<lenght>`是css中最常见的`<dimension>`，形式是数字+长度单位，比如`10px`、`1em`

> 注意，百分比不属于`<length>`类型，属性`<length-percent>`

css的长度单位分为两种：**绝对长度单位**和**相对长度单位**

### 绝对长度单位
绝对长度单位与其他任何东西都没有关系，通常被认为总是相同大小。

| 单位 | 名称 | 等价换算 | 
| -- | -- | -- |
| cm | 厘米 | 1cm = 96/2.54px | 
| mm | 毫米 | 1mm = 1/10cm | 
| Q | 四分之一毫米 | 1Q = 1/40cm | 
| in | 英寸 | 1in = 2.54cm = 96px |
| pc | 十二点活字 | 1pc = 1/16in | 
| pt | 点 | 1pt = 1/72in | 
| px | 像素 | 1px = 1/96in | 

这些值更多的会用于打印而非在页面显示，我们在屏幕显示比较常用的是`px`

### 相对长度单位
相对长度单位相对于其他东西，比如父元素字体大小、视窗大小。

使用相对单位的好处是，经过规划，可以让页面在不同尺寸大小的屏幕上都有着较好的展示形式。

下面列出了一些web开发中最常用的一些相对单位。

| 单位 | 相对于 |
| -- | -- |
| em | 在`font-size`中使用是相对于父元素字体的大小，在其他属性中使用则是相对于自身大小，比如`width`
| ex | 字符`x`的高度
| ch | 数字`0`的宽度
| rem | 根元素的字体大小 
| lh | 元素的`line-height`
| vw | 视窗宽度的1%
| vh | 视窗高度的1%
| vmin | 视窗最小尺寸的1%
| vmax | 视窗最大尺寸的1%
 
## 百分比`<length-precent>`
百分比是有别于`<length>`的css数据类型。

很多属性都支持长度或百分比，也有些只支持长度，注意区分。

百分比值通常是相对于父元素的，比如设置字体大小是相对于父元素字体大小，设置宽度是相对于父元素宽度

## 数字
纯数字类型很少，比如`opacity`，只需要数值不需要单位

## 颜色
颜色有以下写法：
1. 16进制：`#c55d51`，可以简写，比如：`#fff`
1. 关键字：`white` 见[`<color>`](https://developer.mozilla.org/zh-CN/docs/Web/CSS/color_value)
1. rgb：`rgb(2, 121, 139)` 设置的分别是红、绿、蓝
1. rgba：`rgba(18, 138, 125, .9)` 设置的分别是红、绿、蓝、透明度
1. hsl：`hsl(188, 97%, 28%)` 设置的分别是色调、饱和度、亮度
1. hsla：`hsla(174, 77%, 31%, .9)` 设置的分别是色调、饱和度、亮度、透明度

## 标识符
在颜色一节中我们看到可以使用关键字作为值，有很多地方会使用关键字，比如`position`,`background-repeat`这些

## 字符串
有一些允许将字符串作为值，比如：
```css
content: "这是一个字符串，它在css中被引用"
```
